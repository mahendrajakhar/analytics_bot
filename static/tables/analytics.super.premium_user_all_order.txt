Table: analytics.super.premium_user_all_orders
Description: Contains comprehensive information about all orders from premium users.
Primary Key: order_id (appears to be unique).

Foreign Keys (Potential/Inferred):
final_profile_id -> users.id
plan_id -> plans.id
coupon_code_id -> coupons.coupon_code_id
subscription_id -> subscriptions.subscription_id
prev_plan_id -> plans.id

Relationships:
One-to-many with users.
One-to-many with plans.
One-to-many with inferred coupons.
One-to-many with inferred subscriptions.
Self-referencing via prev_order_tz, and prev_plan_id, to track changes in the plans.

Implicit/Joinable Fields:
order_tz, plan_start_date, plan_end_date, etc.: Dates.
primary_lang, language, source, country, platform_type, etc.: Dimensions.
spend_campaign_name and sing_network: Could link back to biz_logic_costs or inferred campaign tables.
source links with biz_logic_costs.source and free_trial_orders.source
